# 👨‍💻 Luiz Gutemberg - Página Pessoal

Website pessoal com design retrô desenvolvido em HTML e CSS, com o objetivo de apresentar informações acadêmicas, projetos e interesses na área de tecnologia.

---

## 🌐 Acesse o site

👉 https://luizgtberg.github.io/luizgtberg/

---

## 🎯 Sobre o projeto

Este projeto foi desenvolvido como uma forma de criar uma presença online simples e personalizada, utilizando apenas HTML e CSS.

O design foi inspirado na estética da web retrô, trazendo elementos clássicos como:
- ASCII art
- cores neon (verde/terminal)
- layout minimalista

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  

---

## 📌 Funcionalidades

- Apresentação pessoal  
- Seção “Sobre mim”  
- Projetos e experiências acadêmicas  
- Contato com links diretos  
- Estilo visual retrô com animações simples
